# Drive bundle — папки команд

Здесь лежат заготовки папок команд для заливки на Google Drive.

## Генерация

Структура собирается скриптом из `scripts/make-drive-bundle.mjs` по списку команд из `site/lib/data/teams.ts`:

```bash
node scripts/make-drive-bundle.mjs
```

После запуска появится:

- `Команды трека ИИ/` — папка с 8 подпапками (по команде).
- В каждой подпапке: `README.md`, `team-guide.md`, `module-template.md`, `resources/`.

Также можно упаковать архивом:

```bash
cd drive-bundle && zip -r teams-drive-bundle.zip "Команды трека ИИ"
```

## Как залить на Drive

1. Откройте Google Drive (или папку саммита).
2. Создайте папку **«Команды трека ИИ»** (или используйте существующую папку саммита).
3. Откройте папку и перетащите в браузере папку `Команды трека ИИ` (или содержимое `drive-bundle/Команды трека ИИ/*`).
4. Drive сохранит вложенность — 8 подпапок по командам.
5. Выделите каждую подпапку команды → «Поделиться» → «Все, у кого есть ссылка» → «Редактор».
6. Скопируйте прямую ссылку на каждую папку и разошлите капитанам команд.

## Что внутри папки команды

- `README.md` — короткая инструкция с составом команды и первыми шагами.
- `team-guide.md` — полный гайд для учителей.
- `module-template.md` — шаблон модуля со всеми пятью разделами (открываем через Google Docs).
- `resources/` — для картинок, скриншотов, ноутбуков.

## Обновление

Если изменили `team-guide.md` или `module-template.md` в `site/public/project/` — перезапустите скрипт, он перегенерирует все папки.
